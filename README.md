# 1D - 3 Points Arc

Blender add-on.

Add-on functionality
-
Creates arc with specified number of points from 2 edges (3 start points).

One corner point on the edge must be selected to detect what vertices will be used for creating the arc.

Current version
-
1.1.0.

Blender version
-
2.79
